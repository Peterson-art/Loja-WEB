# Loja Virtual - Truetech

Projeto full stack de loja virtual com cadastro, login, listagem de produtos e pedidos.

## Funcionalidades
- Cadastro de usuários
- Login
- Catálogo de produtos
- Carrinho
- Registro de pedidos
- Área administrativa

## Tecnologias
- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
